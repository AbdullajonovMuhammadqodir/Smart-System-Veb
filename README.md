# 🚀 Smart System - Task Management & Personal Productivity Platform

**Smart System** is a modern web-based ecosystem designed to organize daily workflows and tasks. Beyond a simple "To-Do" list, this project serves as a comprehensive startup foundation, featuring planned AI assistance and Pro-subscription models.

---

## 🌟 Key Features
* **Secure Authentication:** User data is safely managed using an SQLite database.
* **Personal Dashboard:** A customized interface driven by unique User Nicknames.
* **Task Management:** Full CRUD capabilities to add, view, and manage daily tasks.
* **Modern UI/UX:** Built with Glassmorphism and Dark Mode aesthetics for a premium feel.
* **Fully Responsive:** Optimized for a seamless experience across mobile, tablet, and desktop.

---

## 🛠 Tech Stack
* **Backend:** Python (Flask Framework)
* **Database:** SQLite3
* **Frontend:** HTML5, CSS3, JavaScript
* **Icons:** FontAwesome 6
